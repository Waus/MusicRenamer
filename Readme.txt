Put this program into a directory in which you want to automatically rename files with a pattern:
XX - Title
where XX is a number starting from 01.
This also works in all subdirectories.

This only works if the original file name is one of these most common patterns:
XX Title
XX. Title
XX-Title
XX- Title
X Title
X. Title
X-Title
X- Title

Examples:
02 Hey hey Oh!  ----->  02 - Hey hey Oh!
02. Hey hey Oh!  ----->  02 - Hey hey Oh!
02-Hey hey Oh!  ----->  02 - Hey hey Oh!
02- Hey hey Oh!  ----->  02 - Hey hey Oh!
2 Hey hey Oh!  ----->  02 - Hey hey Oh!
2. Hey hey Oh!  ----->  02 - Hey hey Oh!
2-Hey hey Oh!  ----->  02 - Hey hey Oh!
2- Hey hey Oh!  ----->  02 - Hey hey Oh!
